# 纪念日功能说明

## 功能概述
纪念日功能允许情侣记录和管理重要的日期，包括恋爱纪念日、生日、结婚纪念日等。

## 页面结构

### 1. 纪念日列表页面 (`anniversary.vue`)
- **路径**: `/pages/anniversary/anniversary`
- **功能**: 
  - 显示所有纪念日列表
  - 统计总纪念日数量、即将到来的纪念日、已过的纪念日
  - 支持添加新纪念日
  - 支持编辑和删除现有纪念日
  - 支持下拉刷新

### 2. 纪念日详情页面 (`detail.vue`)
- **路径**: `/pages/anniversary/detail`
- **功能**:
  - 查看纪念日详细信息
  - 编辑纪念日信息
  - 删除纪念日

## 主要功能

### 添加纪念日
1. 在纪念日列表页面点击右上角的"+"按钮
2. 填写纪念日名称、选择类型、设置日期、添加描述
3. 点击保存完成添加

### 编辑纪念日
1. 在纪念日列表页面点击编辑按钮
2. 修改相关信息
3. 点击保存完成编辑

### 删除纪念日
1. 在纪念日列表页面点击删除按钮
2. 确认删除操作

### 查看纪念日详情
1. 点击纪念日列表中的任意项目
2. 进入详情页面查看完整信息

## 纪念日类型
- **恋爱纪念**: 恋爱相关的纪念日
- **生日**: 生日纪念
- **结婚纪念**: 结婚相关的纪念日
- **其他**: 其他类型的纪念日

## 倒计时显示
- **就是今天**: 纪念日当天
- **明天**: 纪念日前一天
- **还有X天**: 7天内的纪念日
- **还有X周**: 30天内的纪念日
- **还有X个月**: 更长时间的纪念日

## API接口
- `getMemorialDayList()`: 获取纪念日列表
- `getMemorialDay(id)`: 根据ID获取纪念日详情
- `addMemorialDay(data)`: 添加纪念日
- `updateMemorialDay(data)`: 更新纪念日
- `deleteMemorialDay(data)`: 删除纪念日

## 数据格式
```javascript
{
  "ID": 1,
  "CreatedAt": "2025-08-19T00:38:14+08:00",
  "UpdatedAt": "2025-08-19T00:38:16+08:00",
  "user_id": 1,
  "lover_id": 2,
  "title": "恋爱纪念日",
  "date": "2025-02-19"
}
```

### 字段说明
- `ID`: 纪念日唯一标识
- `title`: 纪念日名称
- `date`: 纪念日期（格式：YYYY-MM-DD）
- `user_id`: 创建用户ID
- `lover_id`: 情侣ID
- `CreatedAt`: 创建时间
- `UpdatedAt`: 更新时间

## 注意事项
1. 纪念日日期格式为 YYYY-MM-DD
2. 倒计时会自动计算今年的纪念日，如果已过则计算明年的
3. 支持下拉刷新获取最新数据
4. 所有操作都有相应的成功/失败提示
