# 情侣菜谱小程序 - 用户使用说明

## 前言

该小程序最初只是为了我和女朋友使用的，所以很多功能没有考虑完善，大家可以自行扩展 也欢迎fork和提PR。
该小程序后端基于`gin-vue-admin`框架开发，前端基于uniapp开发。
后端基础功能和代码完善，如有需要可以自行添加管理后台功能，地址见下方。

[gin-vue-admin地址 ](https://github.com/flipped-aurora/gin-vue-admin)  https://github.com/flipped-aurora/gin-vue-admin

由于日志功能在个人小程序中无过审，需要企业备案，所以大家自行斟酌要不要删除，想保留日志（日记）功能可以使用体验版。


## 后端项目启动指南

### 环境准备

- Go 1.20 及以上版本
- PostgreSQL 12 及以上版本
- Redis 5.0 及以上版本（可选）

### 项目克隆

```bash 
git clone https://github.com/Echo7659/Couple-s-Menu.git
cd Couple-s-Menu/server
```

### 配置修改

- 修改`config.yaml`中的数据库配置(pgsql)，无需自动建表，程序会自动建表
- 如需使用Redis，修改redis配置
- 修改aliyun-oss配置（可以自行更换）
- 小程序SDK配置 （AppID，AppSecret），修改`utils/wechat/mini.go`中的对应配置


## 前端项目启动指南

前端我这里使用的是`HBuilderX`，如果没有安装请先安装`HBuilderX`，并安装`uni-app`插件。

### 进入到miniAPP目录

```bash 
cd ../miniAPP
```

### 安装依赖

```bash
npm install
```

### 修改配置

- 修改`src/config/index.js`中的后端API地址
- 修改`manifest.json`中的小程序AppID
- 修改`src/config/index.js`中的小程序AppID

### 运行项目

使用`HBuilderX`打开`miniAPP`目录，选择运行到微信小程序模拟器，等待编译完成即可预览。



## 📱 应用简介

**情侣菜谱**是一款专为情侣设计的小程序应用，帮助你们记录美好时光、管理菜谱、完成打卡目标，让每一餐都充满爱意。

### 主要功能

- 💕 **情侣绑定**: 邀请另一半一起使用，共同记录美好时光
- 📝 **日志记录**: 记录你们的日常点滴和美好回忆
- 🍽️ **菜谱管理**: 创建专属菜谱，方便点餐下单
- 🎯 **打卡目标**: 制定小目标，一起完成打卡
- 📅 **纪念日**: 记录重要纪念日，不错过每一个特殊时刻

## 📖 使用指南

### 一、首次使用 - 登录与注册

#### 1.1 打开小程序

打开微信，搜索"小呆呆的私人菜谱"小程序，或通过朋友分享的链接进入。

> ![image-20251130152744354](/images/imgimgimgimage-20251130152744354.png)

#### 1.2 微信登录

1. 进入小程序后，点击 **"微信一键登录"** 按钮
2. 授权微信登录，允许获取你的微信信息

> ![image-20251130152755778](/images/imgimgimage-20251130152755778.png)

#### 1.3 完善个人信息

登录成功后，系统会引导你完善个人信息：

**步骤1: 设置头像**

- 点击头像区域
- 选择"从相册选择"或"拍照"
- 选择或拍摄一张满意的头像照片

> ![image-20251130152834992](/images/imgimgimage-20251130152834992.png)

**步骤2: 填写昵称**

- 在昵称输入框中输入你的昵称
- 可以输入中文、英文或数字

**步骤3: 选择角色**

- **饲养员**: 负责点餐下单的一方
- **吃货**: 享受美食时光的一方
- 选择一个角色，点击卡片即可选中

**步骤4: 保存信息**

- 确认头像、昵称、角色都填写完整后
- 点击 **"保存"** 按钮
- 系统会自动保存你的信息

---

### 二、邀请另一半 - 情侣绑定

#### 2.1 分享邀请

1. 完成个人信息设置后，进入首页
2. 如果还没有绑定情侣，右侧会显示一个 **"+"** 图标
3. 点击右侧头像区域（带"+"的圆圈）
4. 在弹出的分享菜单中，选择 **"发送给朋友"** 或 **"分享到朋友圈"**

> ![image-20251130152919149](/images/imgimgimage-20251130152919149.png)

#### 2.2 对方接受邀请

1. 你的另一半通过你分享的链接进入小程序
2. 对方完成登录和基本信息设置
3. 系统会自动识别邀请码并绑定情侣关系

#### 2.3 查看情侣信息

绑定成功后，首页会显示：

- 左侧：你的头像和昵称
- 中间：爱心图标
- 右侧：另一半的头像和昵称
- 下方：恋爱天数统计

> ![image-20251130152939930](/images/imgimgimage-20251130152939930.png)

---

### 三、设置恋爱开始日期

#### 3.1 修改恋爱天数

1. 在首页找到 **"我们在一起已经 X 天"** 的区域
2. 点击该区域，会弹出日期选择器
3. 选择你们开始恋爱的日期
4. 点击 **"确定"** 保存

> ![image-20251130152946359](/images/imgimgimage-20251130152946359.png)

#### 3.2 查看恋爱天数

设置完成后，首页会自动计算并显示：

- 你们在一起的天数
- 开始恋爱的日期

> ![image-20251130153025501](/images/imgimgimage-20251130153025501.png)

---

### 四、纪念日管理

#### 4.1 添加纪念日

1. 在首页找到 **"重要纪念日"** 区域
2. 点击右上角的 **"+"** 按钮
3. 填写纪念日信息：
    - **纪念日名称**: 如"第一次见面"、"生日"等
    - **日期**: 选择纪念日的日期
    - **类型**: 选择纪念日类型（恋爱纪念日/生日/其他）
4. 点击 **"保存"** 完成添加

> ![image-20251130153036775](/images/imgimgimage-20251130153036775.png)

#### 4.2 查看纪念日

- 首页会显示最近的3个纪念日
- 每个纪念日卡片显示：
    - 纪念日名称
    - 日期
    - 倒计时（还有X天/就是今天/已过）

#### 4.3 查看所有纪念日

1. 点击任意纪念日卡片
2. 进入纪念日详情页面
3. 可以查看所有纪念日，并进行编辑或删除

---

### 五、写日志

#### 5.1 创建日志

**方式一: 从首页创建**

1. 在首页找到 **"写日志"** 卡片
2. 点击进入日志创建页面

**方式二: 从日志页面创建**

1. 点击底部导航栏的 **"日志"** 标签
2. 点击右下角的 **"+"** 按钮

#### 5.2 编辑日志内容

1. **填写标题**: 为日志起一个标题
2. **编写内容**: 在内容区域写下你想记录的文字
3. **添加图片**:
    - 点击图片上传区域
    - 选择"从相册选择"或"拍照"
    - 可以添加多张图片
4. **选择心情**: 选择当前的心情（可选）
5. 点击 **"发布"** 保存日志

#### 5.3 查看日志

1. 进入 **"日志"** 页面
2. 可以看到所有已发布的日志
3. 点击任意日志可以查看详情
4. 可以给日志点赞

---

### 六、菜谱管理

#### 6.1 创建菜谱

1. 点击底部导航栏的 **"菜谱"** 标签
2. 点击 **"新增菜谱"** 卡片
3. 填写菜谱信息：
    - **菜谱名称**: 如"周末美食"、"工作日简餐"等
    - **菜谱描述**: 可选，描述这个菜谱的用途
    - **封面图片**: 可选，上传一张封面图
4. 点击 **"创建"** 完成

> ![image-20251130153103046](/images/imgimgimage-20251130153103046.png)

#### 6.2 管理菜谱

- **查看菜谱**: 点击菜谱卡片进入菜谱详情
- **编辑菜谱**: 长按菜谱卡片，选择"编辑"
- **删除菜谱**: 长按菜谱卡片，选择"删除"

> ![image-20251130153113330](/images/imgimgimage-20251130153113330.png)



#### 6.3 添加菜品分类

1. 进入菜谱详情页面
2. 点击 **"分类管理"** 按钮
3. 填写菜品信息：
    - 分类名称
4. 点击 **"保存"** 完成添加

> **📸![image-20251130153159584](/images/imgimgimage-20251130153159584.png)

#### 6.4 添加菜品

1. 进入菜谱详情页面
2. 点击 **"添加菜品"** 按钮
3. 填写菜品信息：
    - **菜品名称**
    - **价格**（可选）
    - **图片**（可选）
    - **描述**（可选）
4. 点击 **"保存"** 完成添加

> ![image-20251130153238262](/images/imgimgimage-20251130153238262.png)

---

### 七、点餐下单

#### 7.1 选择菜谱

1. 在首页点击 **"点餐下单"** 卡片
2. 或从 **"菜谱"** 页面选择要使用的菜谱
3. 进入点餐页面

> ![image-20251130153308621](/images/imgimgimage-20251130153308621.png)

#### 7.2 选择菜品

1. 浏览菜谱中的菜品
2. 点击想要点的菜品
3. 可以调整数量
4. 选好后点击 **"确认订单"**

#### 7.3 确认订单

1. 检查订单信息：
    - 所选菜品
    - 数量
    - 总价
2. 可以添加备注
3. 点击 **"提交订单"** 完成下单

> ![image-20251130153317325](/images/imgimgimage-20251130153317325.png)

#### 7.4 查看订单

1. 进入 **"我的"** 页面
2. 点击 **"我的订单"**
3. 可以查看所有历史订单
4. 可以查看订单详情和状态

> ![image-20251130153332954](/images/imgimgimage-20251130153332954.png)

---

### 八、打卡目标

#### 8.1 创建打卡目标

1. 点击底部导航栏的 **"打卡"** 标签
2. 点击 **"新建目标"** 按钮
3. 填写目标信息：
    - **目标名称**: 如"一起运动30天"、"每天说晚安"等
    - **目标描述**: 详细说明目标内容
4. 点击 **"创建"** 完成

> ![image-20251130153341987](/images/imgimgimage-20251130153341987.png)

#### 8.2 进行打卡

1. 进入打卡目标详情页面
2. 点击 **"打卡"** 按钮
3. 可以添加打卡内容、图片等
4. 点击 **"提交"** 完成打卡

> ![image-20251130153401097](/images/imgimgimage-20251130153401097.png)

#### 8.3 查看打卡记录

1. 在目标详情页面
2. 可以查看打卡日历
3. 查看所有打卡记录
4. 查看打卡统计

#### 8.4 完成目标

- 当目标完成后，目标状态会变为 **"已完成"**
- 可以在 **"已完成"** 标签中查看所有已完成的目标

![image-20251130153426775](/images/imgimgimage-20251130153426775.png)

### 九、个人中心

#### 9.1 查看个人信息

1. 点击底部导航栏的 **"我的"** 标签
2. 在顶部可以看到：
    - 头像
    - 昵称
    - 用户ID

> ![image-20251130153436132](/images/imgimgimage-20251130153436132.png)

#### 9.2 编辑个人信息

1. 点击头像上的相机图标，可以更换头像
2. 点击右上角的编辑图标，可以修改昵称和角色

#### 9.3 切换角色

1. 在首页，点击你的角色标签（"饲养员"或"吃货"）
2. 确认切换角色
3. 角色会立即更新

![image-20251130153452131](/images/imgimgimage-20251130153452131.png)

#### 9.4 订单通知

1. 进入 **"我的"** 页面
2. 点击 **"订单通知"**
3. 订阅订单状态提醒
4. 可以查看剩余通知次数（受小程序限制，最多一次性只能订阅6次）



#### 9.5 退出登录

1. 在 **"我的"** 页面底部
2. 点击 **"退出登录"** 按钮
3. 确认退出
4. 退出后需要重新登录

## ❓ 常见问题

### Q1: 如何重新绑定情侣？

**A**: 目前不支持重新绑定，如需更换绑定对象，请联系客服。

### Q2: 可以修改恋爱开始日期吗？

**A**: 可以，点击首页的恋爱天数区域即可修改。

### Q3: 日志可以删除吗？

**A**: 可以，在日志详情页面可以删除自己发布的日志。

### Q4: 菜谱可以分享吗？

**A**: 可以，在菜谱详情页面可以分享给朋友。

### Q5: 打卡目标可以修改吗？

**A**: 可以，在目标详情页面可以编辑目标信息。

### Q6: 如何切换角色？

**A**: 在首页点击你的角色标签即可切换。

### Q7: 忘记登录了怎么办？

**A**: 小程序使用微信登录，重新打开小程序即可自动登录。

### Q8: 数据会丢失吗？

**A**: 所有数据都保存在云端，不会丢失。但建议定期备份重要内容。
